Scoreboard country flags
========================
The GeoIP.dat file must be in your etpub folder to be able to use this feature.

You can get/update GeoIP.dat from the following URL:
http://geolite.maxmind.com/download/geoip/database/GeoLiteCountry/GeoIP.dat.gz
On linux, you can use the bash script update.sh
On windows, don't forget you have to decompress it.
